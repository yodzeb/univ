Exercice 2: Exploitation d'une injection SQL

Sur la page suivante, exploitez le CVE-2014-8295 afin :
  • De déterminer le nombre de colonnes de la requete
  • De sortir le mot de passe root de MySL
  • Ecrire d'un fichier sur le disque '/tmp/VotreNom'
  • Lecture d'un fichier sur le disque '/etc/passwd', ou '/tmp/VotreNom'
  • Ressources:
    • MySQL SELECT..INTO
    • Bacula source code
  • Bonus: utiliser cette seconde faille pour un acces shell au serveur: LFI

Exercice 5: DNS Amplification
